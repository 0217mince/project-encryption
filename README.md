# 项目加密

### 项目介绍

现在很多的项目都是部署在封闭的容器环境中，为了保证项目的安全性，我们需要对项目进行加密，这样可以保证项目的安全性，防止项目盗用部署。

* 使用非对称加密技术
* 容器环境中获取环境变量，例如：容器的IP地址、端口号、数据库连接信息等

### 项目问题

无法连接外网的情况下，没有办法做到十分全面的安全，只能尽可能的增大破解难度。

### 实现思路

1. 通过非对称加密生成两队密钥，A对和B对
2. 通过A对加密项目的配置文件
3. 通过B对生成签名文件
4. 项目启动时，通过B对验证签名文件，通过A对解密配置文件
5. 项目启动时，通过环境变量获取容器的IP地址、端口号、数据库连接信息等
6. 解密后配置文件和环境变量对比，如果一致则启动项目，否则退出程序
